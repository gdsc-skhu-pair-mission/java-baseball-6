# 숫자야구

## 컴퓨터 - 랜덤 숫자 생성
- 3자리 수
- 1~9까지의 숫자 
- 중복 불가

## 플레이어 - 입력
- 예외처리
  - 3자리 자연수가 아닌 경우
  - 1~9까지의 숫자가 아닌 경우
  - 중복된 값이 존재하는 경우
- 게임이 끝난 경우 재시작/종료를 구분하는 1과 2 중 하나의 수

## 비교
- 같은 숫자, 같은 자리 = 스트라이크
- 같은 숫자, 다른 자리 = 볼
- 다른 숫자, 다른 자리 = 낫싱 

## 출력
- 볼, 스트라이크 개수 표시
- 3스트라이크 = 게임 종료 후 재시작/종료 선택 

## 종료 
- 1을 누를 경우 = 재시작
- 2를 누를 경우 = 종료
